# Data Structures in C++

This repository contains my personal and academic implementations of fundamental Data Structures from scratch using C++. 

## About the Project
As an Information Systems Engineering student pursuing a double major in Computer Engineering and a minor in Artificial Intelligence, I created this repository to demonstrate my understanding of core computer science concepts, algorithm logic, and software architecture.

## Features & Technologies
* **Language:** C++
* **Core Concepts Applied:**
  * Object-Oriented Programming (OOP)
  * Template Classes (for generic data types)
  * Dynamic Memory Allocation & Pointer Management
* **Implemented Structures:**
  * `LL_h.cpp` - A fully functional Singly Linked List with traversal, insertion, deletion, and searching algorithms.
